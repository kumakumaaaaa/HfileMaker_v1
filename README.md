# HfileMaker (Nursing Assessment System)

入院患者の「一般病棟用の重症度、医療・看護必要度」を日次で評価・記録し、月間推移を管理するWebアプリケーションです。

## 起ち上げ方法

```bash
# 開発サーバーの起動
npm run dev
```
ブラウザで `http://localhost:3000` にアクセスしてください。

---

## 要件定義・仕様概要

### 1. 患者管理
*   左サイドバーに患者リストを表示（リサイズ可能）。
*   選択した患者の月間評価データをメイン画面に表示。

### 2. 月間評価マトリクス (Monthly Matrix View)
*   **マトリクス表示**: 縦軸に項目（A/B/C）、横軸に日付（1日〜月末）を配置。
*   **固定表示 (Sticky)**: スクロール時も「項目名」「日付ヘッダー」「判定行」が常に追従。
*   **状態可視化**: 重症度基準を満たす日はヘッダーと判定行を赤色で強調。

### 3. 編集・入力機能
*   **編集モード**: 日付ヘッダーの「編集（鉛筆）」ボタンでその日のみ編集可能。
    *   **未来日の制限**: 未来の日付は編集不可。
*   **入力操作**:
    *   マウス: クリックでON/OFF切り替え。
    *   キーボード: テンキー(0,1,2)や矢印キーで高速入力可能。
*   **未保存ガード**: 編集中のまま移動・リロードしようとすると警告ダイアログを表示。

### 4. 便利機能
*   **前日コピー**:
    *   未入力の日（データなし）の場合のみ、ヘッダーに「コピー」ボタンを表示。
    *   前日のデータをコピーして即座に編集を開始できる。
*   **データ削除**:
    *   編集モード中、列の下部に「ゴミ箱」アイコンを表示。クリックでその日のデータを削除。
*   **変更インジケータ**:
    *   編集中、元のデータと値が異なる箇所にのみ「赤い点」を表示（新規/コピー時は全項目）。

### 5. データ保存
*   ブラウザの `LocalStorage` にデータを保存（デモ用仕様）。
